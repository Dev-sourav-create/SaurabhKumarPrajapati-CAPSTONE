# 🍕 Pizzeria - React Capstone Project

A responsive pizza ordering application built using **React**, **Redux Toolkit**, **React Router**, and **Tailwind CSS**.

## Features

- Browse pizzas
- Search pizzas
- Filter by Veg / Non-Veg
- Sort pizzas by price
- Build your own custom pizza
- Shopping cart
- Quantity management
- Bill summary
- Responsive design
- Redux global state management
- Redux Persist (Cart persistence)

---

# Tech Stack

- React 19
- Vite
- Redux Toolkit
- React Redux
- Redux Persist
- React Router DOM
- Tailwind CSS

# Installation

Clone the repository

```bash
git clone <repository-url>
```

Go inside the project

```bash
cd <project-name>
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```

Build for production

```bash
npm run build
```

Preview production build

```bash
npm run preview
```

---

# Required Packages

Install the following packages if starting from scratch.

## React

```bash
npm install react react-dom
```

## Vite

```bash
npm create vite@latest
```

## React Router

```bash
npm install react-router-dom
```

## Redux Toolkit

```bash
npm install @reduxjs/toolkit react-redux
```

## Redux Persist

```bash
npm install redux-persist
```

## Tailwind CSS

```bash
npm install tailwindcss @tailwindcss/vite
```
